### Clone & Adapt Protocol

- Clone the closest existing route, server function, Drizzle query, component, dialog, table, form, or admin page before inventing a new pattern.
- Keep marketplace, admin, auth, listing, order, payment, payout, package, and RBAC flows structurally consistent.
- Change resource-specific names, schemas, labels, columns, filters, and mutations only.
- Extract shared behavior instead of copy-pasting it a third time.
- Never edit generated route-tree output manually.
